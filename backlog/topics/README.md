Topics are stored here under `TOPIC-YYYYMMDD_HHMMSS-<slug>/`.

Each topic contains:
- `topic.json`, `brief.json` (or `BRIEF-*.json`), `PDCA.json`, `test-plan.json`, `links.json`
- `tasks/<TASK-ID>/` with `task.json`, `PDCA.json`, `log.jsonl` (and optional `log.md`)

See `BACKLOG_VISION.md` for the full backlog standard.
